# Text

The Text component renders a rich-text block using the Jahia CKEditor integration.

## Properties

| Field | Description |
|-------|-------------|
| Text | Rich-text content edited inline with CKEditor |

The editor uses a compact toolbar (`Tinny`) suited for body content.

## Notes

- The HTML produced by CKEditor is rendered directly. Standard Bootstrap typographic styles apply automatically.
- For layout, wrap Text components inside a Grid.
