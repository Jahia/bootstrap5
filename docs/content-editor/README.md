# Content Editor Guide

This guide is for content editors who create and manage page content in Jahia using the Bootstrap 5 component library.

## Available components

| Component | Description |
|-----------|-------------|
| [Accordion](components/accordion.md) | Collapsible content panels |
| [Alert](components/alert.md) | Contextual notification banners |
| [Breadcrumb](components/breadcrumb.md) | Automatic page hierarchy navigation |
| [Button](components/button.md) | Links and interactive triggers (modal, collapse, popover, offcanvas) |
| [Card](components/card.md) | Flexible content containers with image, header, body, footer |
| [Carousel](components/carousel.md) | Image slideshow |
| [Figure](components/figure.md) | Image with optional caption |
| [Grid](components/grid.md) | Page layout — sections, containers, rows, columns |
| [Navbar](components/navbar.md) | Site navigation header |
| [Pagination](components/pagination.md) | Page-by-page content navigation |
| [Tabs](components/tabs.md) | Tabbed content panels |
| [Text](components/text.md) | Rich-text content block |

## How to add a component

1. In edit mode, click the **+** button inside a droppable area
2. Browse or search the component picker
3. Select a Bootstrap 5 component
4. Fill in its properties in the Content Editor panel on the right
5. Publish when ready

## Color vocabulary

Many components share the same color palette derived from Bootstrap 5 contextual colors:

| Value | Meaning |
|-------|---------|
| `primary` | Brand color |
| `secondary` | Muted accent |
| `success` | Positive / green |
| `info` | Informational / blue |
| `warning` | Caution / yellow |
| `danger` | Error / red |
| `light` | Near-white |
| `dark` | Near-black |
